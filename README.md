# PulseBar

<img src="iconikai-icon-pack/ios/AppIcon-1024.png" width="128" alt="PulseBar 图标">

PulseBar 会把 Mac 当前播放的声音变成实时频谱，显示在顶部菜单栏。播放音乐或视频时，频谱会跟着声音变化；没有声音时，则显示安静状态。

**[下载最新版 PulseBar](https://github.com/pw22286/PulseBar/releases/latest)**

## 功能

- 四种频谱样式
- 横向或竖向显示
- 可调整宽度、方向和颜色
- 柱状频谱支持峰值悬停
- 支持登录时启动
- 支持 Intel 和 Apple Silicon Mac

## 安装

1. 在 Releases 页面下载最新的 DMG。
2. 打开 DMG，把 PulseBar 拖进“应用程序”。
3. 首次启动时，允许“屏幕与系统音频录制”权限，然后重新打开 PulseBar。

需要 macOS 13 Ventura 或更高版本。如果系统提示无法验证开发者，请前往“系统设置 > 隐私与安全性”，点击“仍要打开”。
