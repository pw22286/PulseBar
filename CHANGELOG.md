# Changelog

## 1.11.0

- 山峰样式前层由 55–500 Hz 低频驱动
- 山峰样式后层由 500 Hz–16 kHz 中高频驱动

## 1.10.1

- 修复柱状样式进入静音后低柱连接成横条的问题
- 静音时按当前频谱密度显示一一对应的小圆点

## 1.10.0

- 完成四种真实 FFT 频谱样式与静态样式预览
- 增加横竖方向、伸展方向、宽度和颜色设置
- 增加柱状频谱峰值悬停与样式对应的无音频状态
- 修复菜单栏宽阔档的柱间距和峰值点越界
- 修复登录启动状态处理
- 增加睡眠唤醒恢复和音频流自动重连
- 静音时将 FFT 刷新频率从 30 FPS 降至 8 FPS
- 增加通用架构 DMG、可选 Developer ID 公证和 GitHub Actions 发布流程
